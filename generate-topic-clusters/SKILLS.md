# Generate Topic Clusters

## Description
This skill generates SEO topic clusters by identifying a main pillar topic and related subtopics to create comprehensive content strategies.

## Usage
Use this skill when you need to:
- Plan content around a central topic
- Identify related subtopics for pillar-cluster content strategy
- Build topical authority in a specific niche

## Inputs
- Main topic or keyword
- Target audience (optional)
- Industry/niche context (optional)

## Outputs
- Primary pillar topic
- List of cluster topics
- Suggested internal linking structure
